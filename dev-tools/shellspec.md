# ShellSpec (`shellspec`)

## 基本信息
- 官方文档：https://github.com/shellspec/shellspec
- 安装方式：`brew install shellspec` 或 `git clone https://github.com/shellspec/shellspec.git && make install`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.28

## 核心命令示例
```bash
# 初始化项目
shellspec init

# 运行所有测试
shellspec

# 使用文档格式输出
shellspec --format documentation

# 运行指定测试文件
shellspec spec/*.sh
```

## 适用场景
- Shell脚本的BDD测试框架
- Shell脚本单元测试
- DevOps脚本质量保障
