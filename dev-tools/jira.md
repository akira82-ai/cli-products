# Jira CLI (`jira`)

## 基本信息
- 官方文档：https://github.com/ankitpokhrel/jira-cli
- 安装方式：`brew install jira-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 初始化配置
jira init

# 创建 Issue
jira issue create

# 列出 Issue
jira issue list

# 查看 Sprint 列表
jira sprint list
```

> `jira init` 需提供 Jira 实例地址和 API Token。

## 适用场景
- Jira Issue 创建与管理
- Sprint 追踪与进度查看
- 终端内看板操作
