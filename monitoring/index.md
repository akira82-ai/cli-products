# 监控/可观测

共收录 5 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Better Stack CLI](betterstack.md) | `betterstack` | 管理 Better Stack Uptime 监控和 Heartbeat |
| [Grafana CLI](grafana.md) | `grafanactl` | 管理 Grafana 插件的安装、更新和卸载 |
| [Loki CLI](loki.md) | `logcli` | 在终端中快速查询和过滤 Loki 日志，无需打开 Grafana |
| [Prometheus CLI](prometheus.md) | `promtool` | CI/CD 中自动校验 Prometheus 配置和规则文件的正确性 |
| [Sentry CLI](sentry.md) | `sentry-cli` | CI/CD 中自动上传 Debug Symbols 和 Source Maps |
