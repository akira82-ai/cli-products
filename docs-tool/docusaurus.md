# Docusaurus CLI (`docusaurus`)

## 基本信息
- 官方文档：https://docusaurus.io/docs/cli
- 安装方式：npx create-docusaurus@latest
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.7

## 核心命令示例
```bash
# 创建新站点
npx create-docusaurus@latest mysite classic

# 启动开发服务器
npm run start

# 构建站点
npm run build

# 部署到 GitHub Pages [需认证]
npm run deploy
```

## 适用场景
- 开源项目文档站点
- 版本化文档管理
- React 驱动的文档网站
