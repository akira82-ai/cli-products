# 文档工具

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Alex](alex.md) | `alex` | 检查文档中不包容用语，让你的开源项目更友好 |
| [DocToc](doctoc.md) | `doctoc` | 一键为 Markdown 文档生成 GitHub 风格的锚点目录 |
| [Docusaurus CLI](docusaurus.md) | `docusaurus` | Facebook 出品的文档站生成器，React + MDX 驱动 |
| [Glow](glow.md) | `glow` | 在终端美美地渲染 Markdown，README 查看不再平淡 |
| [Marp CLI](marp.md) | `marp` | 用 Markdown 写 PPT，开发者做演示不再痛苦 |
| [MkDocs CLI](mkdocs.md) | `mkdocs` | Python 生态的静态文档站生成器，Material 主题好评 |
| [Translate Shell](translate-shell.md) | `trans` | 终端里的 Google 翻译，支持 100+ 语言即时翻译 |
