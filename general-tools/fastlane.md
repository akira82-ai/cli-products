# Fastlane (`fastlane`)

## 基本信息
- 官方文档：https://docs.fastlane.tools/
- 安装方式：brew install fastlane
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：2.225

## 核心命令示例
```bash
# 初始化 Fastlane
fastlane init

# 构建并上传到 TestFlight [需认证]
fastlane beta

# 发布到 App Store [需认证]
fastlane release

# 运行测试
fastlane test
```

## 适用场景
- iOS/Android 自动化构建发布
- App Store 和 TestFlight 管理
- 截图和元数据管理
