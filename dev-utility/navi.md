# Navi (`navi`)

## 基本信息
- 官方文档：https://github.com/denisidoro/navi
- 安装方式：brew install navi
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：2.24

## 核心命令示例
```bash
# 打开交互式命令速查表
navi

# 添加社区速查表仓库
navi repo add denisidoro/cheats

# 搜索指定关键词的命令
navi query "docker"

# 直接匹配最佳命令并执行
navi --best-match "git push"
```

## 适用场景
- 交互式命令速查表
- 学习新工具命令
- 替代 tldr 的交互式体验
