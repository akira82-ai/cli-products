# tldr (`tldr`)

## 基本信息
- 官方文档：https://github.com/tldr-pages/tldr
- 安装方式：brew install tldr
- 开源：是 (Creative Commons)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.2

## 核心命令示例
```bash
# 查看 tar 命令的简化用法
tldr tar

# 查看 git log 的常用示例
tldr git log

# 更新本地 tldr 页面缓存
tldr --update

# 查看帮助
tldr --help
```

## 适用场景
- 快速查命令用法
- 替代 man 页简化阅读
- Agent 快速查阅命令示例
