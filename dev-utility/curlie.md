# Curlie (`curlie`)

## 基本信息
- 官方文档：https://github.com/rs/curlie
- 安装方式：brew install curlie
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.6

## 核心命令示例
```bash
# 发送 GET 请求
curlie https://api.example.com

# 发送 POST 请求
curlie POST https://api.example.com name=Alice

# 查看响应头
curlie -I https://example.com

# 查看帮助
curlie --help
```

## 适用场景
- curl 的易用封装
- 结构化 HTTP 请求输出
- 替代 curl 的友好方案
