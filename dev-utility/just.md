# Just (`just`)

## 基本信息
- 官方文档：https://github.com/casey/just
- 安装方式：brew install just
- 开源：是 (Apache-2.0, MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.40

## 核心命令示例
```bash
# 列出所有可用任务
just --list

# 执行默认任务
just

# 执行指定任务
just build

# 带详细输出执行测试
just test --verbose
```

## 适用场景
- 现代版 Makefile
- 项目任务统一入口
- 跨平台构建脚本
