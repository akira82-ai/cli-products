# lazygit (`lazygit`)

## 基本信息
- 官方文档：https://github.com/jesseduffield/lazygit
- 安装方式：brew install lazygit
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.44

## 核心命令示例
```bash
# 启动 Git TUI
lazygit

# 打开指定仓库
lazygit -p /path/to/repo

# 查看帮助
lazygit --help
```

## 适用场景
- Git 仓库可视化 TUI 管理
- 快速分支切换和合并
- 暂存区交互式管理
