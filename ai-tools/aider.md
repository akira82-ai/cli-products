# Aider (`aider`)

## 基本信息
- 官方文档：https://aider.chat/docs/
- 安装方式：pip install aider-chat
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux
- 最后验证版本：0.82

## 核心命令示例
```bash
# 启动交互模式 [需认证]
aider

# 直接编辑文件 [需认证]
aider main.py utils.py

# 指定模型 [需认证]
aider --model gpt-4

# Git 提交模式（自动 commit）
aider --auto-commits
```

## 适用场景
- AI 结对编程
- Git 集成的代码编辑
- 多文件同步修改
