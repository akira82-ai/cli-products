# Qwen Code CLI (`qwen-code`)

## 基本信息
- 官方文档：https://github.com/QwenLM/qwen-code
- 安装方式：pip install qwen-code
- 开源：否
- 平台支持：macOS / Linux
- 最后验证版本：0.1

## 核心命令示例
```bash
# 启动交互模式 [需认证]
qwen-code

# 直接提问
qwen-code "explain this function"

# 查看帮助
qwen-code --help
```

## 适用场景
- 阿里通义千问编程助手
- 中文友好的代码生成
- 终端辅助开发
