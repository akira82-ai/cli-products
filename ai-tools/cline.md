# Cline CLI (`cline`)

## 基本信息
- 官方文档：https://github.com/cline/cline
- 安装方式：VS Code 扩展安装
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.3

## 核心命令示例
```bash
# 通过 VS Code 命令面板使用
# Cmd+Shift+P -> "Cline: Open"

# 创建新功能 [需认证]
cline "create a new feature"

# 调试错误
cline "fix the error in this function"
```

## 适用场景
- VS Code 集成的 AI 编程助手
- 自动化开发任务
- 代码生成和重构
