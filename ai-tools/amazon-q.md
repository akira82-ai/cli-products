# Amazon Q CLI (`q`)

## 基本信息
- 官方文档：https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/
- 安装方式：随 AWS Toolkit 安装
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.5

## 核心命令示例
```bash
# 聊天模式 [需认证]
q chat

# 代码转换 [需认证]
q transform

# 代码解释 [需认证]
q explain

# 代码重构 [需认证]
q refactor
```

## 适用场景
- AWS 环境下的 AI 编程辅助
- 代码解释、重构、转换
- 安全扫描和建议
