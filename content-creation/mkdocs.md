# MkDocs CLI (`mkdocs`)

## 基本信息
- 官方文档：https://www.mkdocs.org/user-guide/cli/
- 安装方式：pip install mkdocs
- 开源：是 (BSD-2-Clause)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.6

## 核心命令示例
```bash
# 创建新项目
mkdocs new my-project

# 启动开发服务器
mkdocs serve

# 构建站点
mkdocs build

# 部署到 GitHub Pages [需认证]
mkdocs gh-deploy
```

## 适用场景
- Markdown 项目文档站点
- 技术文档和 API 文档
- GitHub Pages 部署
