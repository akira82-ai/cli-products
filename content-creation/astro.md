# Astro CLI (`astro`)

## 基本信息
- 官方文档：https://docs.astro.build/en/reference/cli-reference/
- 安装方式：npm create astro@latest
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：5.3

## 核心命令示例
```bash
# 创建新项目
npm create astro@latest mysite

# 启动开发服务器
npx astro dev

# 构建站点
npx astro build

# 预览构建结果
npx astro preview
```

## 适用场景
- 内容驱动网站（博客/文档）
- 多框架组件集成
- 零 JS 默认构建
