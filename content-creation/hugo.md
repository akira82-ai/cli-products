# Hugo CLI (`hugo`)

## 基本信息
- 官方文档：https://gohugo.io/commands/hugo/
- 安装方式：brew install hugo
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.141

## 核心命令示例
```bash
# 创建新站点
hugo new site mysite

# 创建新页面
hugo new posts/my-post.md

# 启动开发服务器
hugo server -D

# 构建站点
hugo --minify
```

## 适用场景
- 静态站点快速构建
- 博客和文档站点生成
- 多语言内容管理
