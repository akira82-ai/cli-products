# Alex (`alex`)

## 基本信息
- 官方文档：https://github.com/get-alex/alex
- 安装方式：npm install -g alex
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：11.0

## 核心命令示例
```bash
# 检查单个 Markdown 文件
alex README.md

# 检查所有 Markdown 文件
alex *.md

# 通过管道检查文本
echo "His code is crazy good" | alex
```

## 适用场景
- 检查文章中不当/不包容用语
- 内容审校
- 项目文档语言规范检查
