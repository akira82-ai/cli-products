# Doctoc (`doctoc`)

## 基本信息
- 官方文档：https://github.com/thlorenz/doctoc
- 安装方式：npm install -g doctoc
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：2.2

## 核心命令示例
```bash
# 为单个文件生成目录
doctoc README.md

# 自定义目录标题和最大层级
doctoc <file> --title="目录" --maxlevel 3

# 批量为多个文件生成目录
doctoc ./docs/*.md
```

## 适用场景
- 长文档自动生成目录
- 批量为多个 Markdown 文件生成 TOC
- 项目文档维护
