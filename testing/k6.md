# k6 CLI (`k6`)

## 基本信息
- 官方文档：https://grafana.com/docs/k6/latest/
- 安装方式：brew install k6
- 开源：是 (AGPL-3.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.56

## 核心命令示例
```bash
# 运行负载测试
k6 run script.js

# 指定虚拟用户数和持续时间
k6 run --vus 100 --duration 30s script.js

# 运行并输出到 InfluxDB [需认证]
k6 run --out influxdb=http://localhost:8086 script.js

# 调试模式
k6 run --http-debug script.js
```

## 适用场景
- 性能和负载测试
- API 压力测试
- 自动化性能回归
