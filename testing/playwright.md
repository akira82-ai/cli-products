# Playwright CLI (`playwright`)

## 基本信息
- 官方文档：https://playwright.dev/docs/cli
- 安装方式：npm install -g playwright
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.50

## 核心命令示例
```bash
# 安装浏览器
playwright install

# 运行测试
npx playwright test

# 调试模式
npx playwright test --debug

# 生成测试报告
npx playwright show-report
```

## 适用场景
- 跨浏览器端到端测试
- Web 应用自动化测试
- 测试录制和代码生成
