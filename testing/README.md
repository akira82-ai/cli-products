# 测试工具

共收录 4 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Cypress CLI](cypress.md) | `cypress` | 自带时间旅行的 E2E 测试框架，可视化和自动重试 |
| [k6 CLI](k6.md) | `k6` | JavaScript 写负载测试，Grafana 集成看压测指标 |
| [Playwright CLI](playwright.md) | `playwright` | 微软出品，Chromium/Firefox/WebKit 三引擎同时测试 |
| [ShellSpec](shellspec.md) | `shellspec` | Shell 脚本的 RSpec，BDD 语法写 Shell 测试用例 |
