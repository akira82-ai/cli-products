# 云服务商

共收录 10 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [阿里云 CLI](aliyun.md) | `aliyun` | 阿里云全产品线管理，ECS/OSS/函数计算等 200+ API 覆盖 |
| [AWS CLI](aws.md) | `aws` | AWS 官方 CLI，S3 多段上传、Lambda 部署、IAM 管理一站式搞定 |
| [Azure CLI](az.md) | `az` | 微软云管理利器，支持输出 JSON/Table/TSV 多种格式 |
| [AWS CDK CLI](cdk.md) | `cdk` | 用 TypeScript/Python 写云资源定义，像写应用代码一样建基础设施 |
| [AWS EB CLI](eb.md) | `eb` | 一行命令部署应用到 Elastic Beanstalk，自动处理容量和负载均衡 |
| [EKS CLI](eksctl.md) | `eksctl` | EKS 集群管理的最简方案，一条命令创建生产级 K8s 集群 |
| [Google Cloud CLI](gcloud.md) | `gcloud` | GCP 全家桶 CLI，支持 Cloud Shell 和本地混合使用 |
| [Google Cloud Storage CLI](gsutil.md) | `gsutil` | GCP 对象存储的 rsync，支持跨云同步和并行传输 |
| [AWS SAM CLI](sam.md) | `sam` | Lambda 本地调试 + API Gateway 模拟，Serverless 开发首选 |
| [腾讯云 CLI](tccli.md) | `tccli` | 腾讯云全产品 CLI，支持输出格式化和自动补全 |
