# 云服务商

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [AWS CLI](aws.md) | `aws` | 管理 AWS 资源（EC2、S3、Lambda 等） |
| [Azure CLI](az.md) | `az` | 管理 Azure 资源（虚拟机、存储、网络等） |
| [AWS CDK CLI](cdk.md) | `cdk` | 用代码定义基础设施（Infrastructure as Code） |
| [AWS EB CLI](eb.md) | `eb` | Elastic Beanstalk 应用部署和管理 |
| [Google Cloud CLI](gcloud.md) | `gcloud` | 管理 GCP 资源（Compute Engine、Cloud Functions 等） |
| [Google Cloud Storage CLI](gsutil.md) | `gsutil` | GCS 对象管理（上传、下载、删除） |
| [AWS SAM CLI](sam.md) | `sam` | Serverless 应用开发与构建 |
