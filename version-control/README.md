# 版本控制

共收录 5 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Bitbucket CLI](bitbucket.md) | `bitbucket` | 管理 Bitbucket 仓库和 Pipeline，Atlassian 生态集成 |
| [GitHub Codespaces CLI](codespaces.md) | `gh cs` | 云端 VS Code 开发环境，CLI 一键创建和管理 |
| [Git](git.md) | `git` | 版本控制的基石，分布式设计让分支和合并随心所欲 |
| [GitHub CLI](github.md) | `gh` | GitHub 官方 CLI，PR/Issue/Release 全在终端搞定 |
| [GitLab CLI](gitlab.md) | `glab` | GitHub CLI 的 GitLab 版，MR/Issue/Pipeline 终端管理 |
