# 大厂工具套件

共收录 8 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Fastlane](fastlane.md) | `fastlane` | App 自动化发布流水线，截图、签名、上传 App Store 全自动 |
| [Flutter CLI](flutter.md) | `flutter` | 一套 Dart 代码编译 iOS/Android/Web/Desktop，热重载秒级刷新 |
| [PowerShell](pwsh.md) | `pwsh` | 微软开源的跨平台 Shell，面向对象的管道设计，Windows/Linux/macOS 通用 |
| [React Native CLI](react-native.md) | `react-native` | 用 React 写原生 App，热重载 + 原生组件，JS 社区生态丰富 |
| [simctl](simctl.md) | `simctl` | 命令行操控 iOS 模拟器，自动化 UI 测试和截屏的必备工具 |
| [VS Code CLI](vscode.md) | `code` | 终端秒开 VS Code，支持 diff 对比、安装扩展、远程连接 |
| [winget](winget.md) | `winget` | Windows 版 Homebrew，一行命令安装 Windows 软件 |
| [xcodebuild](xcodebuild.md) | `xcodebuild` | Xcode 命令行入口，CI/CD 中自动化 iOS/macOS 构建和测试 |
