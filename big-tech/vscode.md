# VS Code CLI (`code`)

## 基本信息
- 官方文档：https://code.visualstudio.com/docs/editor/command-line
- 安装方式：随 VS Code 安装
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.96

## 核心命令示例
```bash
# 在当前目录打开 VS Code
code .

# 打开指定文件
code file.py

# 打开指定文件夹
code /path/to/project

# 安装扩展
code --install-extension publisher.extension
```

## 适用场景
- 从终端快速打开编辑器
- 扩展批量安装
- VS Code 远程开发
