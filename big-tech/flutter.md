# Flutter CLI (`flutter`)

## 基本信息
- 官方文档：https://docs.flutter.dev/
- 安装方式：brew install flutter
- 开源：是 (BSD-3-Clause)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.27

## 核心命令示例
```bash
# 创建新项目
flutter create my_app

# 运行项目
flutter run

# 构建 APK/AAR
flutter build apk

# 获取依赖
flutter pub get
```

## 适用场景
- Flutter 跨平台应用开发
- iOS/Android/Web 桌面应用构建
- Widget 调试和性能分析
