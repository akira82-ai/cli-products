# React Native CLI (`react-native`)

## 基本信息
- 官方文档：https://reactnative.dev/docs/environment-setup
- 安装方式：npm install -g react-native-cli
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.76

## 核心命令示例
```bash
# 创建新项目
npx react-native init MyApp

# 运行 Android
npx react-native run-android

# 运行 iOS
npx react-native run-ios

# 启动 Metro bundler
npx react-native start
```

## 适用场景
- React Native 跨平台应用开发
- 开发热重载
- iOS/Android 应用构建
