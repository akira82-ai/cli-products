# winget (`winget`)

## 基本信息
- 官方文档：https://learn.microsoft.com/windows/package-manager/winget/
- 安装方式：Windows 11 内置
- 开源：否
- 平台支持：Windows
- 最后验证版本：1.9

## 核心命令示例
```bash
# 搜索软件
winget search vscode

# 安装软件
winget install Microsoft.VisualStudioCode

# 列出已安装
winget list

# 升级所有
winget upgrade --all
```

## 适用场景
- Windows 软件包管理
- 批量软件安装
- 系统软件版本管理
