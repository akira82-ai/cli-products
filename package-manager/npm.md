# npm (`npm`)

## 基本信息
- 官方文档：https://docs.npmjs.com/
- 安装方式：随 Node.js 安装（`brew install node`）
- 开源：是 (Artistic-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：v10.9.2

## 核心命令示例
```bash
# 初始化新项目
npm init -y

# 安装依赖包
npm install express

# 全局安装工具
npm install -g typescript

# 运行 package.json 中的脚本
npm run build

# 发布包到 npm registry [需认证]
npm publish

# 查看过时依赖
npm outdated
```

## 适用场景
- Node.js 项目的依赖管理和脚本运行
- 发布和共享 JavaScript/TypeScript 开源包
- 全局 CLI 工具的安装和管理
