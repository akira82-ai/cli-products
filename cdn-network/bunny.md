# Bunny CLI (`bunny-cli`)

## 基本信息
- 官方文档：https://github.com/niclas/bunny-cli
- 安装方式：`npm install -g bunny-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
bunny-cli login

# 清除缓存
bunny-cli purge <url>

# 上传文件到存储
bunny-cli storage upload
```

> `bunny-cli login` 需提供 Bunny.net API Key。

## 适用场景
- Bunny.net CDN 缓存管理
- 存储区域文件上传
- CDN 配置与 Purge 操作
