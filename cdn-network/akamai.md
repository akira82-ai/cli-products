# Akamai CLI (`akamai`)

## 基本信息
- 官方文档：https://techdocs.akamai.com/cli/docs
- 安装方式：`brew install akamai`
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 列出已安装的包
akamai list

# 更新 CLI
akamai update

# 安装扩展包
akamai install <package>

# 列出属性配置
akamai property list
```

> 需要配置 Akamai EdgeRC 凭据文件进行认证。

## 适用场景
- Akamai CDN 属性配置管理
- Edge 逻辑调试与更新
- 扩展包安装与管理
