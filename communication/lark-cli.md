# 飞书 Lark CLI (`lark`)

## 基本信息
- 官方文档：https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/mcp_integration/mcp_introduction
- 官网：https://www.feishu.cn/
- GitHub：https://github.com/larksuite/cli
- 安装方式：`npm install -g @larksuite/cli`
- 开源：是
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
lark-cli auth login

# 查看日历
lark-cli calendar list

# 发送消息
lark-im message send

# 管理文档
lark-doc doc create

# 搜索云空间
lark-doc docs search
```

> `lark-cli auth login` 需先完成飞书账号授权。

## 适用场景
- AI Agent 自动化操作飞书
- 文档与表格管理
- 日历与会议管理
- 消息发送与群聊管理
