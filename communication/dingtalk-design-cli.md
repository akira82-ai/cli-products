# 钉钉 DingTalk Design CLI (`dingtalk-design`)

## 基本信息
- 官方文档：https://open.dingtalk.com/document/download/dingtalk-design-introduction
- 官网：https://www.dingtalk.com/
- 开放平台：https://open.dingtalk.com/
- 安装方式：`npm install -g dingtalk-design-cli`
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 初始化小程序项目
dingtalk-design init miniapp

# 初始化 H5 微应用
dingtalk-design init h5

# 初始化工作台组件
dingtalk-design init component

# 构建项目
dingtalk-design build

# 开发调试
dingtalk-design dev
```

> 使用前需确保 Node.js 版本符合要求（建议 v14+）。

## 适用场景
- 钉钉小程序开发
- H5 微应用开发
- 工作台组件开发
- 前端应用研发
