# Slack CLI (`slack`)

## 基本信息
- 官方文档：https://api.slack.com/automation/cli
- 安装方式：`npm install -g slack`
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
slack auth login

# 发送消息
slack message send

# 创建应用
slack app create

# 列出函数
slack function list
```

> `slack auth login` 等操作需先完成认证。

## 适用场景
- Slack 应用开发与调试
- 消息自动化发送
- 工作流构建与管理
