# Zoom CLI (`zoom`)

## 基本信息
- 官方文档：https://developers.zoom.us/docs/
- 安装方式：`npm install -g zoom-cli`
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# OAuth 认证
zoom oauth

# 创建会议
zoom meeting create

# 列出会议
zoom meeting list

# 列出录制文件
zoom recording list
```

> `zoom oauth` 需先注册 Zoom App 并获取凭据。

## 适用场景
- Zoom 会议创建与管理
- 录制文件管理与下载
- 会议自动化调度
