# 飞书 CLI (`lark`)

## 基本信息
- 官方文档：https://open.feishu.cn/document/cli
- 安装方式：`brew install lark`
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
lark auth login

# 发送消息
lark message send

# 创建文档
lark doc create

# 读取电子表格
lark sheet read
```

> `lark auth login` 等操作需先完成认证。

## 适用场景
- 飞书消息发送与自动化
- 飞书文档和电子表格管理
- 自动化工作流编排
