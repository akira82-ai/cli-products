# Microsoft Teams CLI (`teams`)

## 基本信息
- 官方文档：https://learn.microsoft.com/microsoftteams/platform/
- 安装方式：`npm install -g @microsoft/teams-cli`
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
teams login

# 创建 Teams 应用
teams app create

# 安装应用到团队
teams app install

# 发送消息
teams message send
```

> `teams login` 等操作需先完成认证。

## 适用场景
- Teams 应用开发与调试
- 消息发送自动化
- Teams 应用生命周期管理
