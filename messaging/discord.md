# Discord CLI (`discord`)

## 基本信息
- 官方文档：https://discord.com/developers/docs
- 安装方式：`pip install discord-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
discord login

# 发送消息到指定频道
discord send --channel <id> --message "text"

# 列出服务器
discord server list
```

> `discord login` 需提供 Bot Token 或用户凭据。

## 适用场景
- Discord 消息自动化
- 服务器管理与运维
- 频道内容批量操作
