# 钉钉 CLI (`dingtalk`)

## 基本信息
- 官方文档：https://github.com/open-dingtalk/dingtalk-cli
- 安装方式：`npm install -g dingtalk-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
dingtalk auth login

# 发送消息
dingtalk message send

# 机器人发送消息
dingtalk robot send

# 查看帮助
dingtalk --help
```

> `dingtalk auth login` 等操作需先完成认证。

## 适用场景
- 钉钉消息发送与自动化
- 机器人管理与消息推送
- 自动化通知与告警
