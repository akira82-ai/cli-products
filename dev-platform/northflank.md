# Northflank CLI (`northflank`)

## 基本信息
- 官方文档：https://northflank.com/docs
- 安装方式：npm install -g northflank
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.2

## 核心命令示例
```bash
# 登录 [需认证]
northflank login

# 部署服务 [需认证]
northflank deploy

# 查看服务列表 [需认证]
northflank service list
```

## 适用场景
- Northflank 平台服务管理
- 容器化应用部署
- CI/CD 集成
