# Vercel CLI (`vercel`)

## 基本信息
- 官方文档：https://vercel.com/docs/cli
- 安装方式：npm install -g vercel
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：39.3

## 核心命令示例
```bash
# 登录 [需认证]
vercel login

# 部署项目 [需认证]
vercel deploy

# 本地开发
vercel dev

# 查看环境变量 [需认证]
vercel env ls
```

## 适用场景
- 前端项目和 Serverless 函数部署
- 环境变量和域名管理
- 本地开发与生产环境联调
