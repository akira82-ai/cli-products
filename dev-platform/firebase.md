# Firebase CLI (`firebase`)

## 基本信息
- 官方文档：https://firebase.google.com/docs/cli
- 安装方式：npm install -g firebase-tools
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：13.29

## 核心命令示例
```bash
# 登录 [需认证]
firebase login

# 初始化项目
firebase init

# 部署 [需认证]
firebase deploy

# 启动本地模拟器
firebase emulators:start
```

## 适用场景
- Firebase 项目部署和管理
- 本地模拟器开发和调试
- Hosting / Functions / Firestore 部署
