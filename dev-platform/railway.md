# Railway CLI (`railway`)

## 基本信息
- 官方文档：https://docs.railway.app/guides/cli
- 安装方式：npm install -g @railway/cli
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.14

## 核心命令示例
```bash
# 登录
railway login

# 初始化项目 [需认证]
railway init

# 部署 [需认证]
railway up

# 管理环境变量
railway variables
```

## 适用场景
- 快速部署应用到 Railway
- 环境变量管理
- 日志查看和服务监控
