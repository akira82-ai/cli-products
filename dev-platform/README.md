# 开发者平台

共收录 12 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Clever Cloud CLI](clever.md) | `clever` | 欧洲 PaaS 平台部署 CLI，支持 Docker 和 Node/Python/Go 等运行时 |
| [Firebase CLI](firebase.md) | `firebase` | Google 出品，Cloud Functions 部署、Hosting 托管、实时数据库管理 |
| [Fly.io CLI](fly.md) | `fly` | 应用部署到全球边缘节点，离用户最近的机房运行 |
| [Heroku CLI](heroku.md) | `heroku` | Git Push 即部署的鼻祖，Dyno 模型让部署简单到极致 |
| [Netlify CLI](netlify.md) | `netlify` | 静态站点和 Serverless 函数部署，本地 Dev Server 一键部署 |
| [Northflank CLI](northflank.md) | `northflank` | 一站式容器+数据库+CI/CD 平台的全功能 CLI 管理 |
| [Platform.sh CLI](platform.md) | `platform` | Git 驱动的 PaaS，每次 Push 自动创建预览环境 |
| [Railway CLI](railway.md) | `railway` | 一行命令部署，自动检测运行时，数据库一键 Provision |
| [Render CLI](render.md) | `render` | 现代 Heroku 替代，支持 Web Service/Cron/Redis 等多种服务 |
| [Supabase CLI](supabase.md) | `supabase` | 本地启动 Supabase 全栈环境（Auth/DB/Storage），开发无需连远程 |
| [Vercel CLI](vercel.md) | `vercel` | 前端部署标杆，Next.js 深度集成，Preview 部署秒级生效 |
| [Cloudflare Workers CLI](wrangler.md) | `wrangler` | 在 300+ 城市边缘节点运行代码，KV/R2/D1 一键绑定 |
