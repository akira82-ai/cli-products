# Supabase CLI (`supabase`)

## 基本信息
- 官方文档：https://supabase.com/docs/guides/cli
- 安装方式：brew install supabase/tap/supabase
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：2.2

## 核心命令示例
```bash
# 初始化项目
supabase init

# 启动本地开发环境
supabase start

# 推送数据库迁移 [需认证]
supabase db push

# 生成 TypeScript 类型
supabase gen types typescript
```

## 适用场景
- Supabase 本地开发和调试
- 数据库迁移管理
- 自动生成 TypeScript 类型定义
