# UI Enhancements

**Major Design Overhaul:** 2026-04-04 (commit b49538e)

**Key Features:**
- Custom gradient headers with radial overlays
- Dark/light mode toggle with system preference detection
- Responsive grid layouts for category cards
- Smooth scroll behavior
- Code copy buttons on code blocks
- Back-to-top button for long pages
- Mobile-friendly navigation menu
- External link indicators
- Animation fade-in effects on scroll

**Color Scheme:**
- Primary gradient: `linear-gradient(135deg, #6366f1 0%, #8b5cf6 50%, #ec4899 100%)`
- Dark mode: #0f172a background, #f1f5f9 text
- Light mode: #ffffff background, #0f172a text

**Typography:**
- Inter font for UI text
- JetBrains Mono for code blocks
- Responsive font sizing with clamp()

**Layout Components:**
- Hero section with stats (categories, tools, contributors)
- Search container with backdrop blur
- Category grid with auto-fit columns
- Tool tables with responsive wrappers
- Footer with navigation links

**Why:** Provide a modern, accessible, and visually appealing interface that works well on both desktop and mobile devices.
