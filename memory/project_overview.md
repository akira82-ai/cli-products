# CLI Products Collection

A comprehensive Jekyll-based static site that catalogs CLI (command-line interface) tools across 20 categories with 170+ tools.

**Purpose:** Provide a searchable, well-organized knowledge base for developers, DevOps engineers, and technical enthusiasts to discover CLI tools.

**Key Stats:**
- 20 categories (dev-utility, package-manager, ai-tools, container, cicd, etc.)
- 170+ CLI tools documented
- GitHub Pages deployment at https://akira82-ai.github.io/cli-products

**Structure:**
- Category directories (e.g., `ai-tools/`, `dev-utility/`)
- Each tool has a dedicated markdown file with installation instructions and usage examples
- Custom Jekyll layouts (home.html, category.html, default.html)
- `_includes/` for reusable components (footer, hero, stats)
- `assets/` containing CSS, JS, icons, and images
