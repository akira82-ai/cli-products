# 支付/电商

共收录 5 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Braintree CLI](braintree.md) | `braintree` | PayPal 旗下支付网关的本地调试工具，模拟交易和 Webhook |
| [PayPal CLI](paypal.md) | `paypal` | 通过 REST API 管理 PayPal 订单、退款和订阅（无独立 CLI） |
| [Shopify CLI](shopify.md) | `shopify` | Shopify 开发全家桶，本地开发+实时预览+部署一条龙 |
| [Square CLI](square.md) | `square` | Square API 的命令行测试工具，模拟支付和退款无需写代码 |
| [Stripe CLI](stripe.md) | `stripe` | Webhook 转发到 localhost、触发测试事件，支付开发调试神器 |
