# Comfy CLI (`comfy`)

## 基本信息
- 官方文档：https://github.com/Comfy-Org/comfy-cli
- 安装方式：pip install comfy-cli
- 开源：是 (GPL-3.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.0

## 核心命令示例
```bash
# 初始化项目
comfy init

# 安装自定义节点
comfy node install custom-node-name

# 启动 ComfyUI
comfy launch

# 下载模型
comfy model download stability-ai/sdxl-turbo
```

## 适用场景
- ComfyUI 工作流管理
- 自定义节点安装
- AI 图像生成工作流配置
