# 通义千问 CLI (`qwen`)

## 基本信息
- 官方文档：https://help.aliyun.com/zh/qwen/
- 安装方式：pip install dashscope
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.20

## 核心命令示例
```bash
# 配置认证 [需认证]
qwen configure

# 对话推理 [需认证]
qwen chat "你好"

# 模型列表 [需认证]
qwen model list

# 查看帮助
qwen --help
```

## 适用场景
- 阿里通义千问大模型推理
- 国内 AI 平台开发
- Qwen 系列模型操作
