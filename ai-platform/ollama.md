# Ollama CLI (`ollama`)

## 基本信息
- 官方文档：https://github.com/ollama/ollama
- 安装方式：brew install ollama
- 开源：是 (MIT)
- 平台支持：macOS / Linux
- 最后验证版本：0.5

## 核心命令示例
```bash
# 运行模型
ollama run llama3.1

# 拉取模型
ollama pull llama3.1

# 查看已下载模型
ollama list

# 启动 API 服务
ollama serve
```

## 适用场景
- 本地运行大语言模型
- 模型下载和管理
- 本地 AI 开发和测试
