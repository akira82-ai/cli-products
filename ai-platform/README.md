# AI 模型平台

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Comfy CLI](comfy.md) | `comfy` | 管理 ComfyUI 节点式 AI 图像生成工作流，支持远程执行 |
| [Hugging Face CLI](huggingface.md) | `huggingface-cli` | AI 模型的「npm」，一行命令下载/上传 LLaMA、Stable Diffusion 等模型 |
| [百度千帆 CLI](kilo.md) | `kilo` | 管理百度千帆平台上的大模型部署和推理任务 |
| [LM Studio CLI](lm-studio.md) | `lms` | 本地运行 GGUF 格式模型，聊天界面 + CLI 双模式切换 |
| [Ollama CLI](ollama.md) | `ollama` | 一行命令本地跑 Llama、Mistral 等开源模型，像用 Docker 一样简单 |
| [通义千问 CLI](qwen.md) | `qwen` | 本地运行通义千问系列模型，支持量化推理和对话模式 |
| [Replicate CLI](replicate.md) | `replicate` | 像调用 API 一样跑云端开源模型，按使用量计费无需 GPU |
