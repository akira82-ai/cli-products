# Replicate CLI (`replicate`)

## 基本信息
- 官方文档：https://replicate.com/docs/reference/cli
- 安装方式：pip install replicate
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.34

## 核心命令示例
```bash
# 登录 [需认证]
replicate login

# 运行模型推理 [需认证]
replicate run stability-ai/sdxl "a cat wearing a hat"

# 查看模型列表
replicate models list

# 查看帮助
replicate --help
```

## 适用场景
- 运行云端 AI 模型推理
- 图像/音频/视频生成
- 模型版本管理
