# LM Studio CLI (`lms`)

## 基本信息
- 官方文档：https://lmstudio.ai/docs
- 安装方式：随 LM Studio 应用安装
- 开源：否
- 平台支持：macOS / Windows / Linux
- 最后验证版本：0.3

## 核心命令示例
```bash
# 启动本地推理服务器
lms server start

# 查看可用模型
lms model list

# 拉取模型
lms pull llama3.1

# 查看帮助
lms --help
```

## 适用场景
- LM Studio 模型管理和推理
- 本地 LLM 服务搭建
- 模型搜索和下载
