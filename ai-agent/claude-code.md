# Claude Code CLI (`claude`)

## 基本信息
- 官方文档：https://docs.anthropic.com/en/docs/claude-code
- 安装方式：npm install -g @anthropic-ai/claude-code
- 开源：否
- 平台支持：macOS / Linux
- 最后验证版本：1.0

## 核心命令示例
```bash
# 启动交互式会话 [需认证]
claude

# 直接提问
claude "explain this code"

# 非交互模式
claude -p "refactor this function" < input.py

# 查看配置
claude config
```

## 适用场景
- 终端 AI 编程助手
- 代码生成、重构、调试
- Git 操作和项目文件管理
