# OpenCode CLI (`opencode`)

## 基本信息
- 官方文档：https://github.com/opencode-ai/opencode
- 安装方式：go install github.com/opencode-ai/opencode@latest
- 开源：是 (MIT)
- 平台支持：macOS / Linux
- 最后验证版本：0.3

## 核心命令示例
```bash
# 启动交互模式 [需认证]
opencode

# 聊天模式
opencode chat

# 查看帮助
opencode --help
```

## 适用场景
- 开源 AI 编程终端工具（Go 编写）
- TUI 界面交互
- 多 LLM 提供商支持
