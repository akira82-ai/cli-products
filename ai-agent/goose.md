# Goose CLI (`goose`)

## 基本信息
- 官方文档：https://github.com/block/goose
- 安装方式：pip install goose-ai
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux
- 最后验证版本：1.1

## 核心命令示例
```bash
# 启动交互模式 [需认证]
goose session start

# 配置提供商
goose configure

# 运行指令 [需认证]
goose run "create a unit test"

# 查看帮助
goose --help
```

## 适用场景
- Block 出品的 AI 编程代理
- 可扩展的 Toolkit 系统
- 企业级 AI 编程辅助
