# CloudBase AI CLI (`cloudbase`)

## 基本信息
- 官方文档：https://github.com/TencentCloudBase/cloudbase-cli
- 安装方式：npm install -g @cloudbase/cli
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：4.2

## 核心命令示例
```bash
# 登录 [需认证]
cloudbase login

# 初始化项目 [需认证]
cloudbase init

# 部署 [需认证]
cloudbase functions:deploy

# AI 功能调用 [需认证]
cloudbase ai:invoke
```

## 适用场景
- 腾讯云开发平台 CLI
- 云函数和数据库管理
- AI 能力集成
