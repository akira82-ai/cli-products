# AI 编程代理

共收录 12 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Aider](aider.md) | `aider` | 直接在终端与 AI 协作编辑代码，支持 Git 自动提交和 diff 预览 |
| [Amazon Q CLI](amazon-q.md) | `q` | 深度集成 AWS 生态的 AI 助手，理解 IAM 权限和服务配置 |
| [Claude Code CLI](claude-code.md) | `claude` | Anthropic 出品，可在终端直接读文件、跑命令、改代码的 AI Agent |
| [Cline CLI](cline.md) | `cline` | VS Code 插件形态的 AI Agent，支持多文件编辑和终端操作 |
| [CloudBase AI CLI](cloudbase.md) | `cloudbase` | 腾讯云开发一站式 CLI，前后端部署和云函数管理一步到位 |
| [OpenAI Codex CLI](codex.md) | `codex` | OpenAI 出品的沙盒终端 Agent，代码变更可回滚、安全可控 |
| [Gemini CLI](gemini.md) | `gemini` | Google 出品，支持多模态输入（图片+代码），Gemini 2.5 驱动 |
| [GitHub Copilot CLI](github-copilot.md) | `gh copilot` | 忘掉 man 页，用自然语言描述需求直接生成 shell 命令 |
| [Goose CLI](goose.md) | `goose` | Block 开源，可切换多种 LLM 后端，支持工具扩展的 Agent |
| [Kimi CLI](kimi.md) | `kimi` | 月之暗面出品，支持超长上下文窗口，适合大型代码库分析 |
| [OpenCode CLI](opencode.md) | `opencode` | Go 编写的极简终端 AI 编程工具，单二进制文件开箱即用 |
| [Qwen Code CLI](qwen-code.md) | `qwen-code` | 阿里通义千问出品，支持中文代码理解和本土化场景优化 |
