# GitHub Copilot CLI (`gh copilot`)

## 基本信息
- 官方文档：https://docs.github.com/copilot
- 安装方式：gh extension install github/gh-copilot
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：随 gh 版本

## 核心命令示例
```bash
# 命令建议 [需认证]
gh copilot suggest "list all files modified in the last 7 days"

# 命令解释 [需认证]
gh copilot explain "git rebase -i HEAD~5"

# 查看帮助
gh copilot --help
```

## 适用场景
- Shell 命令智能建议
- 复杂命令解释和理解
- 终端操作效率提升
