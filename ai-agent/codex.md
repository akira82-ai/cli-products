# OpenAI Codex CLI (`codex`)

## 基本信息
- 官方文档：https://github.com/openai/codex
- 安装方式：npm install -g @openai/codex
- 开源：否
- 平台支持：macOS / Linux / Windows
- 最后验证版本：0.1

## 核心命令示例
```bash
# 启动交互模式 [需认证]
codex

# 直接提问
codex "fix the bug in app.js"

# 查看帮助
codex --help
```

## 适用场景
- 终端 AI 编程代理
- 代码修复和生成
- 自动化开发任务
