# Gemini CLI (`gemini`)

## 基本信息
- 官方文档：https://github.com/google-gemini/gemini-cli
- 安装方式：npm install -g @anthropic-ai/claude-code
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.0

## 核心命令示例
```bash
# 启动交互模式 [需认证]
gemini

# 直接提问
gemini "create a react component"

# 查看帮助
gemini --help
```

## 适用场景
- Google AI 终端编程助手
- 多模态代码生成
- 项目文件分析和编辑
