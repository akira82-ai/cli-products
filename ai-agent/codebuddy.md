# CodeBuddy CLI (`codebuddy`)

## 基本信息
- 官方文档：https://github.com/TencentCloudBase/CodeBuddy
- 安装方式：npm install -g @cloudbase/codebuddy
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux
- 最后验证版本：1.0

## 核心命令示例
```bash
# 启动交互模式 [需认证]
codebuddy

# 聊天模式
codebuddy chat

# 查看帮助
codebuddy --help
```

## 适用场景
- 腾讯云 AI 编程助手
- 终端辅助开发
- 与腾讯云服务集成
