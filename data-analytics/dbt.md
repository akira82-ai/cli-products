# dbt CLI (`dbt`)

## 基本信息
- 官方文档：https://docs.getdbt.com/reference/dbt-commands
- 安装方式：pip install dbt-core
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：1.9

## 核心命令示例
```bash
# 初始化项目
dbt init my_project

# 运行模型
dbt run

# 运行测试
dbt test

# 生成文档
dbt docs generate
```

## 适用场景
- SQL 数据转换和建模
- 数据质量测试
- 文档自动生成
