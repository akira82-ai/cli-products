# 数据分析

共收录 8 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Apache Airflow CLI](airflow.md) | `airflow` | Python 定义 DAG 的任务调度平台，数据工程标配 |
| [BigQuery CLI](bigquery.md) | `bq` | Google 出品的 PB 级数据仓库，标准 SQL 查询 |
| [Databricks CLI](databricks.md) | `databricks` | Apache Spark 商业化平台，Notebook + 集群一键管理 |
| [dbt CLI](dbt.md) | `dbt` | 用 SQL 写数据转换，自带文档和测试，数据工程师的编译器 |
| [Apache Flink CLI](flink.md) | `flink` | 有状态流处理引擎，毫秒级延迟实时计算 |
| [Apache Kafka CLI](kafka.md) | `kafka-*` | 分布式事件流平台，百万级 TPS 消息吞吐 |
| [Snowflake CLI](snowflake.md) | `snow` | 云原生数据仓库，存算分离，按秒计费弹性伸缩 |
| [Apache Spark CLI](spark.md) | `spark-submit` | 大数据处理的事实标准，内存计算比 Hadoop MapReduce 快百倍 |
