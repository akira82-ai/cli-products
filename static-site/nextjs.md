# Next.js CLI (`next`)

## 基本信息
- 官方文档：https://nextjs.org/docs/app/api-reference/cli
- 安装方式：npx create-next-app@latest
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：15.1

## 核心命令示例
```bash
# 创建新项目
npx create-next-app@latest myapp

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 适用场景
- React 全栈应用开发
- SSR/SSG 静态站点生成
- API Routes 开发
