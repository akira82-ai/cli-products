# 邮件/营销

| 工具 | 命令 | 说明 |

| [Postmark CLI](postmark.md) | `postmark` | 专注事务性邮件的高送达率服务，API 驱动 |
| [Resend CLI](resend.md) | `resend` | 开发者友好的邮件服务，React Email 模板支持 |
| [SendGrid CLI](sendgrid.md) | `sendgrid` | Twilio 旗下邮件服务，日处理 10 亿+ 邮件 |
| [Twilio CLI](twilio.md) | `twilio` | 短信/语音/邮件/视频通讯 API，自动化通知首选 |
