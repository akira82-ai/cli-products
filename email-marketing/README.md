# 邮件/营销

| 工具 | 命令 | 说明 |

| [AWS SES](aws-ses.md) | `aws ses` | 通过命令行批量发送事务性邮件 |

| [Postmark CLI](postmark.md) | `postmark` | 高送达率的事务性邮件发送 |
| [Resend CLI](resend.md) | `resend` | 面向开发者的现代事务性邮件服务 |
| [SendGrid CLI](sendgrid.md) | `sendgrid` | 批量发送事务性邮件和营销邮件 |
| [Twilio CLI](twilio.md) | `twilio` | 发送短信、语音通知和邮件的自动化脚本 |
