# HashiCorp 套件

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Boundary CLI](boundary.md) | `boundary` | 零信任远程访问，SSH/RDP/K8s 统一入口，无需 VPN |
| [Consul CLI](consul.md) | `consul` | 服务网格 + 服务发现 + KV 存储，微服务架构核心组件 |
| [Nomad CLI](nomad.md) | `nomad` | 轻量级工作负载编排器，容器 + 裸金属 + 虚拟机统一调度 |
| [Packer CLI](packer.md) | `packer` | 一次配置生成 AWS/GCP/Azure/VBox 多平台镜像 |
| [Vagrant CLI](vagrant.md) | `vagrant` | Vagrantfile 定义开发环境，`vagrant up` 一键拉起虚拟机 |
| [Vault CLI](vault.md) | `vault` | 动态生成数据库凭证，密钥自动轮转，DevOps 安全基座 |
| [Waypoint CLI](waypoint.md) | `waypoint` | Build/Deploy/Release 一条龙，K8s/Docker/Nomad 多后端 |
