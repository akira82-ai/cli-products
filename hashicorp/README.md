# HashiCorp 套件

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Boundary CLI](boundary.md) | `boundary` | 安全远程访问基础设施 |
| [Consul CLI](consul.md) | `consul` | 服务发现和注册 |
| [Nomad CLI](nomad.md) | `nomad` | 工作负载编排（容器和非容器） |
| [Packer CLI](packer.md) | `packer` | 多平台机器镜像自动构建 |
| [Vagrant CLI](vagrant.md) | `vagrant` | 可复现的开发环境 |
| [Vault CLI](vault.md) | `vault` | 密钥和凭据管理 |
| [Waypoint CLI](waypoint.md) | `waypoint` | 应用构建-部署-发布一体化工作流 |
