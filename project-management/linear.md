# Linear CLI (`linear`)

## 基本信息
- 官方文档：https://github.com/linear/linear-cli
- 安装方式：`npm install -g linear-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
linear auth

# 创建 Issue
linear issue create

# 列出 Issue
linear issue list

# 列出项目
linear project list
```

> `linear auth` 需提供 Linear API Key。

## 适用场景
- Linear Issue 管理与追踪
- 项目进度查看
- 自动化工作流集成
