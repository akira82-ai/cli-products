# Asana CLI (`asana`)

## 基本信息
- 官方文档：https://developers.asana.com/docs/
- 安装方式：`pip install asana-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
asana auth

# 列出项目
asana projects list

# 创建任务
asana tasks create

# 列出任务
asana tasks list
```

> `asana auth` 需提供 Asana Personal Access Token。

## 适用场景
- Asana 任务创建与管理
- 项目进度追踪
- 自动化任务分配
