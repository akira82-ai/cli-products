# Monday.com CLI (`monday`)

## 基本信息
- 官方文档：https://developer.monday.com/api-reference
- 安装方式：`npm install -g monday-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
monday auth

# 列出看板
monday boards list

# 创建项目条目
monday items create

# 查询项目条目
monday items query
```

> `monday auth` 需提供 Monday.com API Token。

## 适用场景
- Monday.com 看板管理
- 项目条目批量操作
- 自动化数据同步
