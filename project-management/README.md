# 项目管理

共收录 4 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Asana CLI](asana.md) | `asana` | 终端管理 Asana 任务和项目，适合自动化工作流集成 |
| [Jira CLI](jira.md) | `jira` | 终端操作 Jira Issue/Sprint/Board，敏捷团队提效利器 |
| [Linear CLI](linear.md) | `linear` | 极简 Issue 追踪工具的 CLI，开发者的项目管理首选 |
| [Trello CLI](trello.md) | `trello` | 终端操作 Trello 看板和卡片，脚本化任务迁移和自动化 |
