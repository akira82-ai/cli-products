# Notion CLI (`notion`)

## 基本信息
- 官方文档：https://developers.notion.com/
- 安装方式：`pip install notion-cli`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 搜索页面
notion search "query"

# 创建页面
notion page create

# 查询数据库
notion database query

# 获取页面内容
notion page get
```

> 需要 Notion Integration Token 进行认证。

## 适用场景
- Notion 页面创建与管理
- 数据库查询与数据提取
- 文档自动化操作
