# 容器/编排

共收录 12 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Buildah CLI](buildah.md) | `buildah` | 无守护进程构建 OCI 镜像，Rootless 安全，适合 CI 环境 |
| [Docker Compose CLI](docker-compose.md) | `docker compose` | docker-compose.yml 一键编排多服务，本地开发环境标配 |
| [Docker CLI](docker.md) | `docker` | 容器技术的代名词，Build/Ship/Run 一次搞定 |
| [Helm CLI](helm.md) | `helm` | K8s 的 Homebrew，一个 Chart 打包部署整个应用 |
| [K3s](k3s.md) | `k3s` | 5 个二进制文件跑起来的 K8s，边缘计算和 IoT 首选 |
| [K9s](k9s.md) | `k9s` | K8s 的终端 UI，资源浏览和日志查看比 kubectl 直观百倍 |
| [Kaniko](kaniko.md) | `kaniko` | K8s Pod 内构建镜像，无需特权模式，比 Docker-in-Docker 更安全 |
| [kind](kind.md) | `kind` | Docker 容器里跑 K8s 集群，CI 测试环境秒起秒毁 |
| [Kubernetes CLI](kubectl.md) | `kubectl` | K8s 的瑞士军刀，一切集群操作的入口命令 |
| [Minikube](minikube.md) | `minikube` | 本地一键启动 K8s 集群，支持 Dashboard 和 Ingress |
| [Podman CLI](podman.md) | `podman` | Docker 的无守护进程替代，Rootless 运行更安全，命令兼容 |
| [Skopeo CLI](skopeo.md) | `skopeo` | 跨仓库复制镜像、查看镜像层，无需拉取完整镜像 |
