# CI/CD

共收录 11 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [ArgoCD CLI](argocd.md) | `argocd` | K8s 原生 GitOps 持续交付，Git 提交即部署 |
| [Buildkite CLI](buildkite.md) | `buildkite-agent` | Buildkite 自托管 CI Agent，在自有基础设施上跑流水线 |
| [CircleCI CLI](circleci.md) | `circleci` | 本地验证和调试 CircleCI 配置，避免提交后才发现配置错误 |
| [Flux CLI](flux.md) | `flux` | CNCF GitOps 工具，自动同步 Git 仓库状态到 K8s 集群 |
| [GitLab Runner CLI](gitlab-runner.md) | `gitlab-runner` | 注册和管理 GitLab CI Runner，支持 Docker/Shell/K8s 执行器 |
| [Jenkins CLI](jenkins.md) | `jenkins-cli` | 远程触发 Jenkins 构建和管理节点，JAR 包即开即用 |
| [Octopus Deploy CLI](octopus.md) | `octo` | Octopus Deploy 发布自动化，多环境蓝绿/金丝雀部署 |
| [release-it](release-it.md) | `release-it` | 自动更新版本号、生成 Changelog、打 Tag、推 Release 全流程 |
| [Spinnaker CLI](spinnaker.md) | `spin` | Netflix 开源的多云持续交付平台管理 |
| [Tekton CLI](tekton.md) | `tkn` | K8s 原生 CI/CD 框架的命令行入口，管理 Pipeline 和 Task |
| [Travis CI CLI](travis.md) | `travis` | 本地触发 Travis CI 构建，查看日志和加密环境变量 |
