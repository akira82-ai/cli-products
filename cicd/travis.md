# Travis CI CLI (`travis`)

## 基本信息
- 官方文档：https://github.com/travis-ci/travis.rb
- 安装方式：`gem install travis`
- 开源：是 (MIT)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：—

## 核心命令示例
```bash
# 登录认证
travis login

# 列出仓库
travis repos

# 查看分支
travis branches

# 查看构建日志
travis logs
```

> `travis login` 需提供 GitHub Token 进行认证。

## 适用场景
- Travis CI 构建状态查看
- 构建日志远程查看
- 仓库与分支管理
