# 基础设施即代码

共收录 7 个 CLI 工具。

## 工具列表

| 工具 | 命令 | 说明 |
|------|------|------|
| [Ansible CLI](ansible.md) | `ansible` | 无 Agent 的服务器配置管理，YAML Playbook 定义期望状态 |
| [AWS CDK CLI](aws-cdk.md) | `cdk` | 用 TypeScript/Python 定义 AWS 资源，自带智能提示和类型检查 |
| [CloudFormation CLI](cloudformation.md) | `cloudformation` | AWS 原生 IaC 工具，JSON/YAML 模板声明式管理资源 |
| [Crossplane CLI](crossplane.md) | `crossplane` | 用 K8s CRD 管理云资源，GCP/AWS/Azure 统一 K8s API |
| [Pulumi CLI](pulumi.md) | `pulumi` | 用你熟悉的编程语言写基础设施，循环、条件、函数随意用 |
| [Terraform CLI](terraform.md) | `terraform` | IaC 领域的事实标准，HCL 声明式管理多云资源 |
| [OpenTofu CLI](tofu.md) | `tofu` | Terraform 的开源分叉，Linux 基金会托管，社区驱动 |
