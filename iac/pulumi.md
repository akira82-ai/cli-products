# Pulumi CLI (`pulumi`)

## 基本信息
- 官方文档：https://www.pulumi.com/docs/cli/
- 安装方式：brew install pulumi
- 开源：是 (Apache-2.0)
- 平台支持：macOS / Linux / Windows
- 最后验证版本：3.140

## 核心命令示例
```bash
# 创建新项目
pulumi new aws-typescript

# 预览变更
pulumi preview

# 部署 [需认证]
pulumi up

# 查看堆栈列表
pulumi stack ls
```

## 适用场景
- 用 TypeScript/Python/Go 等通用语言管理基础设施
- 多云部署管理
- 基础设施状态管理
