# 贡献指南

感谢你对 CLI Products Collection 的贡献！

## 如何贡献

1. 复制 `template.md` 作为新工具的模板
2. 填写工具信息（基本信息、核心命令示例、适用场景）
3. 将文件放入对应的分类目录
4. 提交 Pull Request

## 文件命名规范

- 小写英文，用 `-` 连接
- 示例：`tencent-cloud.md`、`hugging-face.md`

## 内容质量要求

- **核心命令示例**：最少 3 个，覆盖认证、常用操作、查看帮助
- **验证版本**：在"基本信息"中填写实际测试的版本号
- **需认证的命令**：在注释中标注 `[需认证]`
- **语言**：中文撰写，命令和专有名词保留英文

## PR 审查标准

- 文件命名符合规范
- 命令示例可执行（至少 `--version` 可通过）
- 信息准确，链接可用
- 中文撰写，专有名词保留英文

## 过时内容处理

发现命令过时时请提交 Issue，维护者会定期更新验证版本。

## 新增分类

如需新增分类目录，请在 PR 中同时添加：
1. 新分类目录及其 `README.md`
2. 更新根 `README.md` 的分类目录表格
